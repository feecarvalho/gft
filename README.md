<h2 id="how-to-run">⚙️ How to run</h2>

* 1 - Build the project at `/backend` with:

```
dotnet build
```

* 2 - Run the Presentation Web API project at `/GTF.Presentation` with:

```
dotnet run
```

* 3 - Open the `index.html` file located at: `/frontend`
